void _send_filter_adjustment(int adjustment)#

Send updated attenuation demand to the motion controller.

Calculate positions of individual filters based on a bitmask of the attenuation level, set the parameters on the motion controller and then execute the motion program to move the motors.

The code to set variables through shared memory are inside of an ARM #ifdef fence, so when compiled for x86 it will just do the calculations and print a message.

Parameters:

adjustment – [in] Attenuation levels to change by (can be positive or negative)
